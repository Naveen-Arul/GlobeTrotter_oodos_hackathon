## What We Have Done

### Project Overview
We developed **GlobeTrotter**, a personalized travel planning application designed to simplify the process of creating and managing multi-city travel itineraries. The platform provides users with tools to organize trips, estimate budgets, and visualize their plans interactively.

### Key Features Implemented

1. **User Authentication**
   - Login and signup functionality with email and password validation.
   - Password recovery options for user convenience.

2. **Dashboard**
   - Centralized hub displaying upcoming trips and quick actions.
   - Recommendations for popular destinations.

3. **Trip Management**
   - Create, edit, and delete trips with details like name, dates, and descriptions.
   - Upload optional cover photos for trips.

4. **Itinerary Builder**
   - Add and manage travel stops, including cities and activities.
   - Interactive interface to reorder stops and assign dates.

5. **City and Activity Search**
   - Search functionality to discover cities and activities.
   - Filters for refining search results based on cost, type, and duration.

6. **Budget Estimation**
   - Automatic cost breakdowns for transport, stay, activities, and meals.
   - Visual charts to help users stay within budget.

7. **Itinerary Visualization**
   - Calendar and timeline views for day-wise trip planning.
   - Expandable day views with quick editing options.

8. **Sharing Options**
   - Publicly shareable itineraries with unique URLs.
   - Social media sharing and "Copy Trip" functionality.

9. **User Profile Management**
   - Editable profile fields for personal data and preferences.
   - Saved destinations for quick access.

### Technical Highlights
- **Frontend:** Built with React, TypeScript, and Tailwind CSS for a responsive and dynamic user interface.
- **Backend:** Utilized a relational database for efficient storage and retrieval of user-specific itineraries, stops, and activities.
- **Deployment:** Ensured the application is production-ready with optimized builds and smooth performance.

### Challenges Addressed
- Designed a relational database schema to handle complex travel itineraries.
- Implemented dynamic user interfaces that adapt to various trip flows.
- Focused on creating a seamless user experience with intuitive navigation and interactivity.
